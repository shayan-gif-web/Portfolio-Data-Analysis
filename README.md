# 📊 Portfolio Data Analysis Dashboard

An interactive Power BI dashboard built to analyze portfolio projects across skills, technologies, experience levels, and yearly growth.

## 🔍 Dashboard Pages
- **Page 1 – KPI & Technology:** Total projects, completion rate, technology usage bar chart, status pie chart
- **Page 2 – Skills & Experience:** Skill category breakdown, experience level distribution
- **Page 3 – Projects by Year:** Year-over-year project growth (2023–2025)
- **Page 4 – Insights:** Written analysis and key findings

## 📈 Key Findings
- 80% project completion rate
- Python is the most used technology
- Intermediate-level projects make up 43% of the portfolio
- Consistent growth from 2023 → 2024 → 2025

## 🛠️ Tools Used
- Power BI
- Microsoft Excel
